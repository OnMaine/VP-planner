# API та дані — Война Племён (voynaplemyon.com)

Война Племён — українська/російська версія Tribal Wars від InnoGames GmbH.
Сервери (світи) мають формат: `ru{N}.voynaplemyon.com` (напр. `ru93.voynaplemyon.com`).

---

## Публічні API (без авторизації)

### Map-файли

Доступні як `.txt` (plain) і `.txt.gz` (gzip). Базовий URL: `https://{world}.voynaplemyon.com`

| Ендпоінт | Вміст |
|---|---|
| `/map/village.txt(.gz)` | Всі села світу |
| `/map/player.txt(.gz)` | Всі гравці |
| `/map/ally.txt(.gz)` | Всі племена |
| `/map/conquer.txt(.gz)` | Всі захоплення (повна історія) |
| `/map/profile.txt(.gz)` | Профілі гравців |

**Формат village.txt** (CSV, URL-encoded):
```
village_id,name,x,y,player_id,points,rank
```

**Формат player.txt**:
```
player_id,name,ally_id,villages,points,rank
```

**Формат ally.txt**:
```
ally_id,name,tag,members,villages,points,rank
```

### Interface.php (XML відповідь)

| Ендпоінт | Вміст |
|---|---|
| `/interface.php?func=get_config` | Налаштування світу (speed, unit_speed, map_size, нічний бонус) |
| `/interface.php?func=get_unit_info` | Характеристики юнітів (speed, attack, defense, carry) |
| `/interface.php?func=get_building_info` | Характеристики будівель |
| `/interface.php?func=get_conquer&since={unix_timestamp}` | Захоплення за останні 24 год (фільтр по часу) |

**get_config** — ключові поля:
- `speed` — множник швидкості світу (ru100 = 2)
- `unit_speed` — множник швидкості юнітів (ru100 = 0.5)
- `coord.map_size` — розмір карти (ru100 = 1000)
- `night.active`, `night.start_hour`, `night.end_hour`, `night.def_factor` — нічний бонус

**get_unit_info** — по кожному юніту:
- `speed` (хв/поле base), `attack`, `defense`, `defense_cavalry`, `carry`, `pop`, `build_time`

## Дані від гравця (CSV-вигрузка)

Юзер вручну вивантажує дані з гри та надає файл `data.csv`.

**Формат** (заголовок + рядки):
```
Игрок,Координаты,Очки,Копья,Мечи,Топоры,Лазы,ЛК,ТК,Тараны,Каты,Пал,Двор
```

**Відповідність колонок CSV до юнітів гри:**

| CSV | Назва в грі | Speed | Атака | Заг. захист | Захист від кав | Вант. | Ферма |
|---|---|---|---|---|---|---|---|
| Копья | Копейщик | 18 | 10 | 15 | 45 | 25 | 1 |
| Мечи | Мечник | 22 | 25 | 50 | 25 | 15 | 1 |
| Топоры | Топорник | 18 | 43.2* | 10 | 5 | 10 | 1 |
| Лазы | Лазутчик | 9 | 0 | 2 | 1 | 0 | 2 |
| ЛК | Лёгкий кавалерист | 10 | 140.4* | 30 | 40 | 80 | 4 |
| ТК | Тяжёлый кавалерист | 11 | 150 | 200 | 80 | 50 | 6 |
| Тараны | Таран | 30 | 2 | 20 | 50 | 0 | 5 |
| Каты | Катапульта | 30 | 100 | 100 | 50 | 0 | 8 |
| Пал | Паладин | 10 | 150 | 250 | 400 | 100 | 10 |
| Двор | Дворянин | 35 | 30 | 100 | 50 | 0 | 100 |

> *Значення з бонусом технологій (дані з гри). API `get_unit_info` повертає базові значення без tech: Топорник=40, ЛК=130.

**Спеціальні можливості:**
- **Лазутчик** — при атаці виявляється тільки ворожими лазутчиками; розкриває доп. інфо у звіті
- **Таран** — руйнує Стіну, знижуючи її захисний бонус
- **Катапульта** — знищує вибрані будівлі
- **Паладин** — підкріплення з паладином рухається зі швидкістю паладина (10 хв/поле)
- **Дворянин** — знижує лояльність на 20–35 очок; село захоплюється при лояльності ≤ 0

## Що НЕ доступно через API

- Реальні дані по військам в реальному часі (тільки ручна CSV-вигрузка)
- Рухи військ (атаки в дорозі, підтримки)
- Відправка атак програмно (порушення правил гри)
- Аутентифікація без офіційного клієнта
